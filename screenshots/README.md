# Screenshots Directory

This directory contains screenshots of the Pharmacy Revenue Management System.

## Available Screenshots

- `dashboard.png` - Main dashboard with overview cards and charts
- `file-upload.png` - File upload interface with drag & drop
- `analytics.png` - Analytics page with revenue breakdowns
- `unmatched-records.png` - Unmatched records management
- `recent-uploads.png` - Recent uploads history
- `admin-panel.png` - Admin panel with user management

## Adding Screenshots

To add screenshots to this directory:

1. Take screenshots of your application
2. Save them as PNG files with descriptive names
3. Place them in this directory
4. Update the README.md file to reference the correct image paths

## Image Guidelines

- Use PNG format for better quality
- Keep file sizes reasonable (< 1MB per image)
- Use descriptive filenames
- Ensure screenshots show the application in a clean, professional state
